# DevOps Nginx Project

## Description
Deployed a web server using Nginx on AWS EC2 through EC2 Instance Connect.

## Tools Used
- AWS EC2
- Amazon Linux
- Nginx
- Git & GitHub

## Steps Followed
1. Launched EC2 instance
2. Connected using EC2 Instance Connect
3. Installed and started Nginx
4. Hosted a static web page

## Result
Successfully hosted a website on AWS EC2.
